# update-basic-context

`AI_Context/` 폴더 안의 .md 파일들을 현재 코드베이스 상태에 맞게 순차적으로 업데이트합니다.

## 실행 순서

아래 순서대로 각 파일을 검토하고 업데이트하세요. 각 파일 완료 후 다음 파일로 넘어갑니다.

---

### 1. `AI_Context/progress.md`

현재 작업 상태와 다음 할 일을 최신화합니다.

- 완료된 작업 항목을 체크 또는 완료 표시
- 현재 진행 중인 작업 상태 반영
- 다음 우선순위 작업 목록 갱신
- 최종 업데이트 날짜를 오늘 날짜로 변경

---

### 2. `AI_Context/architecture.md`

전체 스크립트 구조 지도를 최신화합니다.

- 새로 생성된 파일을 해당 섹션에 추가 (경로, 클래스명, 역할 설명 포함)
- 삭제된 파일 항목 제거
- 이동·이름 변경된 파일 경로 수정
- 총 파일 수 및 최종 업데이트 날짜 갱신

---

### 3. `AI_Context/log.md`

변경 이력을 기록합니다.

- 이번 작업에서 생성·수정·삭제한 파일을 아래 형식으로 상단에 추가:

```
### [YYYY-MM-DD] 작업 제목
- **신규**: `경로/파일명.cs` — 설명
- **수정**: `경로/파일명.cs` — 변경 내용 요약
- **삭제**: `경로/파일명.cs` — 삭제 이유
```

---

### 4. `AI_Context/code_rule.md`

코딩 규칙 문서를 검토합니다.

- 실제 코드베이스와 규칙 간 불일치가 발견된 경우에만 수정
- 새로운 패턴이 코드베이스 전반에 정착된 경우 규칙으로 추가
- 변경 없으면 최종 업데이트 날짜만 갱신

---

### 5. `AI_Context/enemy_system_plan.md`

적 AI 시스템 플랜의 진행 현황을 업데이트합니다.

- 완료된 Phase 항목 체크
- 현재 진행 중인 Phase 상태 반영
- 다음 Phase 계획이 확정된 경우 추가
- 변경 없으면 생략 가능

---

## 주의사항

- 각 파일 업데이트 전 반드시 해당 파일을 읽고 시작할 것
- 실제 변경된 내용만 반영할 것 (추측으로 작성 금지)
- 모든 날짜는 오늘 날짜(YYYY-MM-DD)로 기재할 것
